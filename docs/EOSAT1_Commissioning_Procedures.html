C-BOOT-001 - OBC Boot Failure
▼

Trigger: OBC fails to boot

Recovery Steps:

  1. 1. Attempt warm boot (soft reset via CMD_RESET)
  2. 2. Check power to OBC (BUS_V nominal)
  3. 3. If still failing, perform cold boot sequence
  4. 4. Load backup boot parameters from redundant storage
  5. 5. Escalate to flight director if still non-responsive
Commands:
func_id 55
Success Criteria:
OBC_STATUS=OPERATIONAL

Escalation: Escalate to FD if unsuccessful after 2 cold boots

C-OBDH-001 - OBDH Health Degraded
▼

Trigger: OBDH health parameter drops below threshold

Recovery Steps:

  1. 1. Retrieve error log (func_id 61)
  2. 2. Analyze error codes for root cause
  3. 3. Isolate affected module if identified
  4. 4. Reload module firmware from backup
  5. 5. Re-run health check (func_id 58)
  6. 6. Verify all subsystem links restored
Commands:
func_id 58/61
Success Criteria:
OBDH_HEALTH=OK

Escalation: Continue to next phase if health restored within 10 minutes

C-OBDH-002 - Watchdog Timeout
▼

Trigger: Watchdog timer expires unexpectedly

Recovery Steps:

  1. 1. Review system logs leading up to timeout
  2. 2. Check for CPU overload or stuck processes
  3. 3. Reset watchdog timer (func_id 59)
  4. 4. Return to safe mode briefly to recover
  5. 5. Resume commissioning sequence
Commands:
func_id 59
Success Criteria:
WATCHDOG=RESET

Escalation: If repeated timeouts occur, return to safe mode

C-EPS-002 - Battery Voltage Low
▼

Trigger: Battery voltage falls below 24V

Recovery Steps:

  1. 1. Reduce power load immediately
  2. 2. Disable non-essential heaters and subsystems
  3. 3. Switch to battery discharge mode
  4. 4. Enable solar charging if sun available
  5. 5. Wait 30 minutes for recovery
  6. 6. If voltage still low, enter sunpoint mode
Commands:
func_id 20
Success Criteria:
BATT_V>=24.0V

Escalation: Monitor voltage closely every 5 minutes
